금융투자은행
> 따로 표기되지 않은 계산은 주(week)에 한번 계산됨.
> 금융투자은행은 국가당 한 개만 존재함.

1. 은행 생성
금융투자은행은 아래 조건들에 의해 생성될 수 있다. 하나의 금융투자은행만 생성될 수 있다.

A. 자본 출자식 은행 생성(법률 - 소유권: 생산 수단 몰수일 경우 불허)
금융투자은행이 존재하지 않는다면 금융투자은행이 생성된다.
금융투자은행 생성 즉시 해당 기업의 주식 50,000주가 발행된다.

공모 주식 수 = 발행 주식 수 * 60 * 0.01 # 해당 수량만큼 주식시장에 주식을 공모한다.
공모 주가 = (국가 GDP * 0.001) / 50000

모든 주식의 공모가 끝나면, 공모가 끝난 주에 주식을 지급하고 돈을 받는다.
남은 주식은 자사주로, 기업이 소유한다.

B. 개인 출자식 은행 생성(법률 - 소유권: 생산 수단 몰수일 경우 불허)
재산 상위 10%의 인구층 중, 금융투자은행이 존재하지 않고, 금융투자은행 생성 욕구가 모든 인구층에 존재하지 않고 해당 인구층 중 기업 생성 심리가 없는 경우, 재산 상위 10%의 인구층 중 랜덤한 인구층 한 개가 금융투자은행 생성 심리가 발생한다.
금융투자은행 생성 심리가 발생한 인구층은 자금을 모은다. ((국가 GDP * 0.001 / 50000) * 20000)만큼의 자금을 모은다면, 금융투자은행이 생성되며 자금은 기업의 보유 현금으로 옮긴다.
금융투자은행 생성 즉시 해당 기업의 주식 50,000주가 발행되며, 자금을 출자한 인구층이 20,000주를 제공받는다. 10,000주는 시장에 공모하며 20,000주는 자사주로 소유한다. 

모든 주식의 공모가 끝나면, 공모가 끝난 주에 주식을 지급하고 돈을 받는다.
남은 주식은 자사주로, 기업이 소유한다.

C. 국가 주도 은행 생성(법률 - 금융: 국영 은행일 경우만 허용)
# 법률 - 금융: 국영 은행 법률이 시행중일 경우만 국가 주도 은행 설립이 가능함.
금융투자은행이 존재하지 않으며,  ((국가 GDP * 0.001 / 50000) * 25000)만큼의 자금이 있으면, 플레이어가 기업을 생성할 수 있으며, 기업 생성 시 자금은 기업의 보유 현금으로 옮긴다.

금융투자은행 생성 즉시 해당 금융투자은행의 주식 50,000주가 발행되며, 국가 소유로 들어간다.

2. 포트폴리오
금융투자은행은 기본적으로 금융 서비스와 주거 서비스를 제공하여 자금을 마련한다.

금융투자은행은 다음을 소유하며 운영한다.
주식
채권
부동산
- 건물 부동산
- 토지 부동산

A. 포트폴리오 비중
기준금리와 여러 변수에 따라 포트폴리오를 조정한다.
보유 현금 비중은 0.3, 포트폴리오 비중은 0.7이다. 부동산은 비중에서 제외된다.

A.1. 포트폴리오 계산 식
감도 파라미터:α=β=10
조정 전 주식 비중=0.7*e^-α(현재 기준금리-0.03)*(15/모든 기업의 평균 PER)
조정 전 채권 비중=0.3*e^β(현재 기준금리-0.03)

주식 비중=조정 전 주식 비중/(조정 전 주식 비중+조정 전 채권 비중)
채권 비중=조정 전 채권 비중/(조정 전 채권 비중+조정 전 주식 비중)

총 주식 가치=(종목별) 주가 * 보유 주식
총 채권 가치=보유한 모든 채권  * 1000

포트폴리오 비중 부족 비율=주식/채권 비중-현재 주식/채권 비중

3. 관리 방법
A. 주식
- 국내 주식의 경우
금융투자은행은 주식시장의 시장 참여자이며 마켓메이커로 활동한다.

기업이 생산하는 상품의 매수/매도주문 비율 점수=((매수 주문/매도 주문)-1)

예측 수요 점수(특정 기업이 생산 시설을 늘릴만한 수요가 있을 때, 그 생산 시설에서 소비할 상품을 계산)=((수정치 매수 주문/매도 주문)-1)
# 기업들의 생산시설 건설 및 재가동 조건이 충족될 때 까지 몇 개의 생산시설을 건설할 수 있을지 계산, 조건이 충족될 때 까지 최대로 건설할 수 있는 생산 시설들의 소비 상품 중 특정 상품을 추가하여 현재 매수 주문에 추가한 후 수정된 매수/매도 주문을 계산

PER 점수=((15/per)-1)

발행 주체 신용도 점수: 0.5(50%)이 중앙값이며 낮을 수록 신용이 좋으며 높을수록 신용이 나쁨.

방향성 심리 점수 = (현재 주가 / 52주 평균 주가) * 0.5

정치적 사건 점수 = 투자하려는 주식의 기업이 속해있는 국가에서 전쟁 발발 시 -0.3, 투자하려는 주식의 기업이 속해있는 국가의 같은 대륙 내의 국가에서 전쟁 발발 시 -0.2, 투자하려는 주식의 기업이 속해있는 국가의 다른 대륙에서 전쟁 발발 시 -0.1, 투자하려는 주식의 기업이 속해있는 국가에서 쿠데타 발발 시 -0.2 # 전쟁이 끝나면 26주간 점수가 점차 회복됨. 쿠데타는 발발 후 26주간 점수가 점차 회복됨.

매매 점수(중앙값 0) = 기업이 생산하는 상품의 매수/매도주문 비율 점수 + 예측 수요 점수 + PER 점수 + (0.5 - 발행 주체 신용도 점수 + 포트폴리오 비중 부족 비율) + 방향성 심리 점수 + 정치적 사건 점수

- 해외 주식의 경우
GDP 성장률 차이 = (대상 국가 연간 GDP 성장률 - 자국 연간 GDP 성장률) * 5
절대적 GDP 성장 점수 = (-0.02 + 해당 국가의 전년 동기 대비 GDP) * 1.2
사회 평균 기술 레벨 점수 = 해당 국가의 사회 평균 기술 레벨 * 0.01
인플레이션 점수 = (0.03 - 해당 국가의 전년 동기 대비 인플레이션) * 1.2
경제 성장 점수 = clamp( (GDP 성장률 차이 + 사회 평균 기술 레벨 점수 + 인플레이션 점수 + 절대적 GDP 성장 점수) / 4 , -0.5, 0.5)

주간 환율 변동률 = |(이번 주 환율 / 지난주 환율) - 1| # 52주 평균 주간 환율 변동률을 계산함.
외환 리스크 점수 = clamp( (0.005 - 52주 평균 주간 환율 변동률) * 100, -0.5, 0.5)

자본 통제 점수 = 투자하려는 국가의 헌법: 경제 체제 - 사회주의 경제, 계획 경제일 경우 -0.8, 법률: 금융 - 국영 은행일 경우 -0.5, 법률: 소유권 - 국유화 금지가 아닐 경우 -1, 투자하려는 국가의 세율 점수: -1 * (증권세율 * 0.3)

국가 매력도 점수 = (경제 성장 점수 * 0.6) + (외환 리스크 점수 * 0.4) + (자본 통제 점수)

기업이 생산하는 상품의 매수/매도주문 비율 점수=((매수 주문/매도 주문)-1)

예측 수요 점수(특정 기업이 생산 시설을 늘릴만한 수요가 있을 때, 그 생산 시설에서 소비할 상품을 계산)=((수정치 매수 주문/매도 주문)-1)
기업들의 생산시설 건설 및 재가동 조건이 충족될 때 까지 몇 개의 생산시설을 건설할 수 있을지 계산, 조건이 충족될 때 까지 최대로 건설할 수 있는 생산 시설들의 소비 상품 중 특정 상품을 추가하여 현재 매수 주문에 추가한 후 수정된 매수/매도 주문을 계산

PER 점수=((15/per)-1)

발행 주체 신용도 점수: 0.5(50%)이 중앙값이며 낮을 수록 신용이 좋으며 높을수록 신용이 나쁨.

방향성 심리 점수 = (현재 주가 / 52주 평균 주가) * 0.5

정치적 사건 점수 = 매수하려는 주식의 기업이 속해있는 국가에서 전쟁 발발 시 -0.3, 매수하려는 주식의 기업이 속해있는 국가의 같은 대륙 내의 국가에서 전쟁 발발 시 -0.2, 매수하려는 주식의 기업이 속해있는 국가의 다른 대륙에서 전쟁 발발 시 -0.1, 매수하려는 주식의 기업이 속해있는 국가에서 쿠데타 발발 시 -0.2 # 전쟁이 끝나면 26주간 점수가 점차 회복됨. 쿠데타는 발발 후 26주간 점수가 점차 회복됨.

매매 점수(중앙값 0) = ((기업이 생산하는 상품의 매수/매도주문 비율 점수 + 예측 수요 점수 + PER 점수 + (0.5 - 발행 주체 신용도 점수 + 포트폴리오 비중 부족 비율) + 방향성 심리 점수 + 정치적 사건 점수) * 0.7) + ((해외 주식의 경우 국가 매력도 점수) * 0.3)

- 매매
기본 주문 규모 = min((보유 현금 * 주식 비중) / (상장된 기업 수(공모중인 종목이 있다면 추가) * 종목 주가), 보유 주식)
변동성에 의한 매수 주문 변동치 = 매매하려는 주식의 기업이 속한 국가와의 52주 평균 환율 변동성 + 매매하려는 주식의 52주 평균 주가 변동성

매수 주문 = (기본 주문 규모 / 2) * (1 + 매매 점수 / 2)

공모 매수 예약 주문 = (기본 주문 규모 / 2)

매도 주문 = (기본 주문 규모 / 2) * (1 - 매매 점수 / 2)

(보유 자금을 기반으로 매수 주문과 공모 매수 예약 주문을 재조정, 소숫점 반내림)

최종 환전 규모 = max(0, (재조정된 매수 주문 * 매수할 해외 주식의 기업이 속한 국가의 외환 환율) - 금융투자은행이 보유한 매수할 해외 주식의 기업이 속한 국가의 외환)
재조정 후의 해외 주식 매수 주문과 공모 매수 예약 주문 = (재조정된 매수 주문) - (재조정된 매수 주문 * 변동성에 의한 매수 주문 변동치)

A.2. 주식 공모
기업이 주식을 공모하면 공모한 주식을 매수 예약함. 이 때 (공모 주가 * 매수 예약 주문 수) 만큼의 자금은 예약되며, 주식 비중에 포함됨. 그 기업의 모든 주식이 매수 예약되면 예약된 자금이 기업으로 보내지며, 공모한 주식을 받음.

B. 채권
금융투자은행은 채권시장의 시장 참여자이며 마켓메이커로 활동한다.

- 해외 채권의 경우
GDP 성장률 차이 = (대상 국가 연간 GDP 성장률 - 자국 연간 GDP 성장률) * 5
절대적 GDP 성장 점수 = (-0.02 + 해당 국가의 전년 동기 대비 GDP) * 1.4
GDP대비 국가 부채 규모 점수 = -1 * GDP대비 국가 부채 규모 * 채권 유동성 계수
채권 유동성 계수 = 1 - (52주 평균 해당 채권의 거래량 / 해당 채권의 총 발행량)
사회 평균 기술 레벨 점수 = 해당 국가의 사회 평균 기술 레벨 * 0.01
인플레이션 점수 = (0.03 - 해당 국가의 전년 동기 대비 인플레이션) * 1.2
경제 성장 점수 = clamp( (GDP 성장률 차이 + 사회 평균 기술 레벨 점수 + 인플레이션 점수 + 절대적 GDP 성장 점수 + GDP대비 국가 부채 규모 점수) / 5 , -0.5, 0.5)

주간 환율 변동률 = |(이번 주 환율 / 지난주 환율) - 1| # 52주 평균 주간 환율 변동률을 계산함.
외환 리스크 점수 = clamp( (0.005 - 52주 평균 주간 환율 변동률) * 100, -0.5, 0.5)

자본 통제 점수 = 투자하려는 국가의 헌법: 경제 체제 - 사회주의 경제, 계획 경제일 경우 -0.8, 법률: 금융 - 국영 은행일 경우 -0.5, 법률: 소유권 - 국유화 금지가 아닐 경우 -1, 투자하려는 국가의 세율 점수: -1 * (증권세율 * 0.3)

국가 매력도 점수 = (경제 성장 점수 * 0.6) + (외환 리스크 점수 * 0.4) + (자본 통제 점수)

발행 주체 신용도 점수: 0.5(50%)이 중앙값이며 낮을 수록 신용이 좋으며 높을수록 신용이 나쁨.

채권 수익률 점수=((해당 채권의 이자율-시장 평균 채권 이자율)/시정 평균 채권 이자율+포트폴리오 비중 부족 비율)

매매 점수(중앙값 0) = ((채권 수익률 점수 + (0.5 - 발행 주체 신용도 점수)) * 0.7) + (해외 채권의 경우 국가 매력도 점수 * 0.3)

기본 주문 규모 = floor(보유 현금 * 채권 비중 / 시장 최고가 채권 가격)
변동성에 의한 매수 주문 변동치 = 매매하려는 채권의 기업이 속한 국가와의 52주 평균 환율 변동성 + 매매하려는 채권의 52주 평균 가격 변동성

매수 주문 = (기본 주문 규모 / 2) * (1 + 매매 점수 / 2)

공모 매수 예약 주문 = (기본 주문 규모 / 2) * (1 + 매매 점수 / 2)

매도 주문 = (기본 주문 규모 / 2) * (1 - 매매 점수 / 2)

(소숫점 반내림)

최종 환전 규모 = max(0, (재조정된 매수 주문 * 매수할 해외 채권의 기업이 속한 국가의 외환 환율) - 금융투자은행이 보유한 매수할 해외 채권의 기업이 속한 국가나 국채의 주체 국가의 외환)
재조정 후의 해외 채권 매수 주문과 공모 매수 예약 주문 = (재조정된 매수 주문) - (재조정된 매수 주문 * 변동성에 의한 매수 주문 변동치)

B.2. 채권 공모

기업이 채권을 공모하면 공모한 채권을 매수 예약함. 이 때 (1000 * 매수 예약 주문 수) 만큼의 자금은 예약되며, 채권 비중에 포함됨. 그 기업의 모든 채권이 매수 예약되면 예약된 자금이 기업으로 보내지며, 공모한 채권을 받음.

C. 외환 매도
해당 주(week)에 채권과 주식 매수를 위해 외환을 추가로 매수해야 할 때는 매도하지 않는다. 채권과 주식 매수에 필요한 외환을 제외한 후에도 보유 외환이 남는다면 아래 식에 따라 외화를 매도한다.

채권과 주식 매수에 필요한 외화를 제외한 보유 외화 매도량 = 총 외화 - 채권과 주식 매수에 필요한 외화

D. 부동산
금융투자은행은 부동산을 매입하여 건물을 건설한다.

D.1. 빌딩 매입
금융투자은행이 소유한 빌딩은 빌딩이 건설된 주에서 주거 서비스 1을 매도하며 수익을 얻는다.

주의 주거 서비스 수요가 공급을 초과할 경우 부동산 시장에서 빌딩을 매입한다. 부동산 시장에서 빌딩 매도 물량이 존재하지 않을 경우 토지를 매입해 직접 빌딩을 건설한다.
빌딩의 건설 완공에 필요한 건설 서비스: 160

주의 주거 서비스 수요가 (주거 서비스 매도 주문) - (주거 서비스 매수 주문) 의 값이 1 이상이라면 빌딩을 부동산 시장에 매도한다.

D.1. 주택 매입
금융투자은행이 소유한 주택은 빌딩이 건설된 주에서 고급 주거 서비스 0.5를 매도하며 수익을 얻는다.

주의 고급 주거 서비스 수요가 공급을 초과할 경우 부동산 시장에서 주택을 매입한다. 부동산 시장에서 주택 매도 물량이 존재하지 않을 경우 토지를 매입해 직접 주택을 건설한다.
주택의 건설 완공에 필요한 건설 서비스: 40

주의 고급 주거 서비스 수요가 (고급 주거 서비스 매도 주문) - (고급 주거 서비스 매수 주문) 의 값이 0.5 이상이라면 주택을 부동산 시장에 매도한다.

E. 건설 및 수리
매주 최대 4의 건설 서비스를 시설이나 건물이 건설되는 주에서 매수하며, 매수한 다음 주에 실제 체결된 건설 서비스 주문만큼 건설 포인트가 증가한다. 건설 포인트가 완공에 필요한 건설 포인트만큼 모였다면 해당 시설이나 건물 건설이 완료된다.
건설이 완료된 건물은 내구도를 가지며, 최대 내구도는 완공에 필요한 건설 포인트만큼이다. 완공 직후는 내구도가 최대치로 설정된다.

건설 기간동안 기업은 건설 서비스 4를 매수하려 한다.
기업은 생산 시설과 사무 시설을 동시에 건설할 수 있다. 단, 생산 시설과 사무 시설은 한번에 각각 2개만 건설할 수 있다.

기업의 총 매주 건설 비용(실제 체결 후 시행되어야 함) = 건설 서비스 가격 * 모든 건설 프로젝트에서 실제 체결된 건설 서비스 수량 * 진행중인 건설 수

만일 시설이 화재나 폭격 등으로 인해 손상을 입을 경우 건물을 수리하려 한다. (현재 시설 내구도 / 최대 시설 내구도) 값이 0.5이하인 경우 해당 시설은 폐쇄된다. 단, 재가동할 수 있는 시설로 간주되어 새로운 시설 건설은 하지 않으나 (현재 시설 내구도 / 최대 시설 내구도)값이 0.5이하인 경우 재가동할 수 없다.

이 경우 매주 최대 4의 건설 서비스를 시설이나 건물을 수리하려는 주에서 매수하며, 매수한 다음 주에 실제 체결된 건설 서비스 주문만큼 내구도가 증가한다. 건설 포인트가 완공에 필요한 건설 포인트만큼 모였다면 해당 시설이나 건물 수리가 완료된다.
매주 수리 비용은 "매주 건설 비용"과 같다.

4. 자금 조달 방법
인구층이 금융투자은행에 돈을 지급한 돈 1당 펀드 상품 1을 받는다. 인구층이 지급한 돈은 보유 현금에 추가되며, 펀드에 따른 분배금은 매 주 (해당 인구층이 투자한 펀드 금액 * (기준금리 * 2) / 52)로 시민에게 지급한다.

5. 재정
금융투자은행은 보유 현금, 수입, 지출, 순수입, 보유 주식, 보유 채권, 보유 부동산, 펀드 규모 정보를 가지고 있다.

A. 주주환원
금융투자은행은 주간 수입을 일정 비율로 주주에게 배당하거나 자사주를 소각한다.
자사주 소각은 기업이 이미 자사주를 소유하고 있으면 소유한 자사주를 소각하고, 자사주가 부족하거나 없다면 시장에서 자사주를 매입하여 소각한다.
배당 성향은 기업 직종에 따라 나뉜다. 주마다 배당금을 주주에게 지급한다.

주(week)간 1주(stock)당 배당금 = (주간 순수입 * 0.3 * 지난주 대비 실적 증가율) / 발행 주식 수
주(week)간 자사주 소각 규모 = (주간 순수입 * 0.4 * 지난주 대비 실적 증가율)
