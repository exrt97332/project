# Python 국가운영 시뮬레이션 디스코드 봇

`Ideas/` 폴더의 설계 문서를 바탕으로 국가 경제/금융 MVP를 상세화한 디스코드 봇입니다.

## 실행

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
export DISCORD_TOKEN=YOUR_TOKEN
python src/sim_bot.py
```

## 명령어

- `!시작 [국가명]` : 새 시뮬레이션 시작
- `!요약` : 핵심 지표 확인
- `!다음주 [주수]` : 주간 루프 진행 (최대 52주)
- `!기준금리 [퍼센트]` : 중앙은행 기준금리 조정
- `!세율 [퍼센트]` : 정부 세율 조정
- `!거시규제 [LTV%] [DSR%]` : LTV/DSR 규제 변경
- `!은행한도 [예대율%] [지준율%]` : 은행 목표 예대율/지급준비율 변경
- `!중앙은행대출 on/off` : 중앙은행 자동 유동성 지원 허용/금지
- `!이벤트` : 최근 이벤트 로그
- `!아이디어` : Ideas 요약 보기
- `!자동 on/off` : 5분 주기 자동 진행

## MVP 상세 구현

- 거시경제 루프
  - 인구층 수입/세금/예금/소비성 현금 흐름
  - 기업 주간 손익, 법인세, 적자 시 자동대출
  - GDP/물가/인플레 주간 반영
- 은행/중앙은행 시스템
  - 기준금리 기반 예금금리·대출금리 자동 계산
  - 예대율/지준율 목표 위반 시 신규대출 제한
  - 중앙은행 자동 유동성 공급(재할인 대출 성격)
  - 금융불안정성 지표 + 뱅크런 플래그 + 파산 상태
- 부동산 담보대출 MVP
  - LTV/DSR 심사
  - 슬리피지 기반 체결가 반영
  - 20년(1040주) 변동금리 상환
  - 채무불이행 시 담보 매각/회수
- 상태 저장
  - 서버(guild)별 JSON 세이브 자동 저장 (`data/sim_<guild_id>.json`)
