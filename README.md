# Python 국가운영 시뮬레이션 디스코드 봇

`Ideas/` 폴더의 설계 문서를 바탕으로, 국가/중앙은행/시중은행/인구/기업/주(State) 중심의 주간 경제 루프를 포함한 디스코드 봇을 구현했습니다.

## 실행

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
export DISCORD_TOKEN=YOUR_TOKEN
python src/sim_bot.py
```

## 명령어

- `!시작 [국가명]` : 새 시뮬레이션 시작
- `!요약` : 핵심 지표 확인
- `!다음주 [주수]` : 주간 루프 진행 (최대 52주)
- `!기준금리 [퍼센트]` : 중앙은행 기준금리 조정
- `!세율 [퍼센트]` : 정부 세율 조정
- `!이벤트` : 최근 이벤트 로그
- `!아이디어` : Ideas 요약 보기
- `!자동 on/off` : 5분 주기 자동 진행

## 구현 범위

- 아이디어 문서의 핵심 항목들을 게임 가능한 형태로 통합:
  - 중앙은행 기준금리 → 은행 예금/대출 금리 연동
  - 인구 예금/대출/채무불이행 처리
  - 기업 손익 및 현금흐름, 자동 대출
  - 정부 세입/지출, 국고 변화
  - 시장 물가 및 인플레이션 지표
  - 은행 금융불안정성 및 뱅크런 발생 로직
  - 주(State) 데이터 초기화
- 서버(guild)별 JSON 세이브 파일 자동 저장
